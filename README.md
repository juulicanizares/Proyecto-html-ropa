# Proyecto-HTML

## Descripción

Este proyecto consiste en el desarrollo de una tienda web estática utilizando HTML y CSS. Su objetivo es simular el flujo completo de compra de un usuario, desde la visualización de productos hasta el proceso de pago.

## Tecnologías utilizadas

- HTML5
- CSS
- Diseño Responsive


# creacion de carpetas del proyecto

Se inició el desarrollo del proyecto mediante la creación de la estructura de carpetas y archivos necesarios para organizar correctamente todos los recursos. Esta estructura permite mantener un orden claro entre archivos HTML, estilos CSS e imágenes, facilitando el mantenimiento y la escalabilidad del proyecto a futuro.

![carpetas](IMG/Carpetas.png)

# creacion de index.html 

Se creó el archivo index.html, el cual corresponde a la página principal del proyecto. Este archivo sirve como punto de entrada para los usuarios y base inicial del funcionamiento del sitio web.

![carpetas](IMG/index.png)

# creacion del catalogo.html

Se desarrolló el archivo catalogo.html, destinado a mostrar los productos disponibles dentro del sitio. Este apartado es fundamental para la navegación del usuario, ya que permite visualizar las opciones de compra.

![carpetas](IMG/catalogo.png)

# creacion del producto.html

Se creó el archivo producto.html, el cual representa la vista individual de cada producto. En esta sección se podrá detallar información específica como descripción, precio y características del producto seleccionado.

![carpetas](IMG/producto.png)

# creacion del carrito.html

Se implementó el archivo carrito.html, encargado de gestionar los productos seleccionados por el usuario. Aquí se podrán visualizar los artículos agregados antes de proceder con la compra.

![carpetas](IMG/carrito.png)

# creacion del checkout.html

Se creó el archivo checkout.html, el cual permite al usuario finalizar el proceso de compra. En esta sección se integrarán los datos necesarios para completar la transacción.

![carpetas](IMG/checkout.png)

# creacion del perfil.html

Se desarrolló el archivo perfil.html, destinado a la gestión de la información del usuario. Este espacio permitirá visualizar y modificar datos personales dentro del sistema.

![carpetas](IMG/perfil.png)

# creacion del historial.html

Se creó el archivo historial.html, el cual permitirá al usuario consultar el registro de sus compras realizadas anteriormente dentro de la plataforma.

![carpetas](IMG/historial.png)

# creacion del vacio.html

Se implementó el archivo vacio.html como plantilla base o estructura inicial para futuras páginas. Este archivo facilita la creación de nuevas secciones manteniendo un diseño consistente.

![carpetas](IMG/vacio.png)


# creacion del producto-clemont.html

En esta sección se muestra el detalle de la camiseta clemont, incluyendo su imagen, precio, descripción y selección de talla antes de agregar el producto al carrito.


![carpetas](IMG/camisaclemont.png)

# creacion del producto-hoodie-clemont.html

Aquí se presenta el hoodie clemont, permitiendo al usuario visualizar sus características, seleccionar la talla y añadirlo al carrito de compras.


![carpetas](IMG/hoodieclemont.png)

# creacion del producto-hoodie-undgld.html

Esta vista corresponde al hoodie UNDGLD, donde se muestra la información del producto junto con opciones de selección de talla y acceso al carrito.


![carpetas](IMG/hoodieundgl.png)


# creacion del producto-undgld.html

En esta página se detalla la camiseta undergold, mostrando su imagen, precio, descripción y opciones para seleccionar la talla antes de realizar la compra.

![carpetas](IMG/camisaundgl.png)

# creacion de base.css

Se creó el archivo base.css, donde se definen los estilos generales del proyecto, como tipografías, colores base y configuraciones globales que serán utilizadas en todas las páginas.

![carpetas](IMG/base.png)

# creacion layout.css

Se desarrolló el archivo layout.css, encargado de la distribución y estructura de los elementos dentro de la página, como encabezados, contenedores y organización del contenido.

![carpetas](IMG/layout.png)

# creacion de components.css

Se creó el archivo components.css, el cual contiene los estilos de componentes reutilizables como botones, tarjetas, formularios y otros elementos comunes dentro del sitio.

![carpetas](IMG/components.png)

# creacion de responsive.css

Se implementó el archivo responsive.css, destinado a adaptar el diseño del proyecto a diferentes tamaños de pantalla, garantizando una correcta visualización en dispositivos móviles, tablets y computadoras.

![carpetas](IMG/responsive.png)
